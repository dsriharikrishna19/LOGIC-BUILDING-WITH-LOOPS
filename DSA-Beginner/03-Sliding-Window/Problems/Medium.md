# 🟡 Medium Sliding Window Problems

These problems introduce variable window techniques and constrained window patterns.

| # | Problem | Concept |
|---|---------|---------|
| 1 | Longest Substring Without Repeating Characters | Variable Window |
| 2 | Longest Repeating Character Replacement | Variable Window |
| 3 | Minimum Size Subarray Sum | Variable Window |
| 4 | Fruit Into Baskets | At Most K Distinct |
| 5 | Max Consecutive Ones III | At Most K Zeros |
| 6 | Frequency of the Most Frequent Element | Sliding Window + Sorting |
| 7 | Number of Subarrays of Size K and Average ≥ Threshold | Fixed Window |
