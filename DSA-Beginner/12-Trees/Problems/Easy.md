| 1 | Maximum Depth of Binary Tree | DFS |
| 2 | Same Tree | Tree Comparison |

| 4 | Symmetric Tree | Mirror DFS |
| 5 | Path Sum | Root-to-Leaf DFS |
| 6 | Minimum Depth of Binary Tree | BFS / DFS |
