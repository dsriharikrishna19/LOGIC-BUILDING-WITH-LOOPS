# 🟡 Medium BST Problems

These problems introduce BST deletion, iterative traversal, and balanced tree construction.

| # | Problem | Concept |
|---|---------|---------|
| 1 | Delete Node in a BST | BST Deletion |
| 2 | Kth Smallest Element in a BST | Inorder Traversal |
| 3 | Convert BST to Greater Tree | Reverse Inorder |
| 4 | BST Iterator | Stack + Inorder |
| 5 | Construct BST from Preorder Traversal | Recursive Construction |
| 6 | Convert Sorted Array to BST | Balanced BST |
| 7 | Trim a Binary Search Tree | BST Pruning |
