# 🟡 Medium Heap Problems

These problems introduce heap-based selection, greedy, and two-heap techniques.

| # | Problem | Concept |
|---|---------|---------|
| 1 | Kth Largest Element in an Array | Min Heap |
| 2 | Top K Frequent Elements | Heap + HashMap |
| 3 | Find K Closest Elements | Heap |
| 4 | K Closest Points to Origin | Max Heap |
| 5 | Reorganize String | Greedy + Max Heap |
| 6 | Task Scheduler | Heap + Greedy |
| 7 | Find Median from Data Stream | Two Heaps |
