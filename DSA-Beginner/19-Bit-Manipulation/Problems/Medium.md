# 🟡 Medium Bit Manipulation Problems

These problems introduce bitmasking, bit prefix, and trie-based bit techniques.

| # | Problem | Concept |
|---|---------|---------|
| 1 | Single Number II | Bit Counting |
| 2 | Bitwise AND of Numbers Range | Bit Prefix |
| 3 | Subsets | Bitmasking |
| 4 | Gray Code | Bit Generation |
| 5 | Maximum XOR of Two Numbers in an Array | Trie + Bits |
| 6 | Minimum Flips to Make a OR b Equal to c | Bit Logic |
| 7 | UTF-8 Validation | Bit Masking |
