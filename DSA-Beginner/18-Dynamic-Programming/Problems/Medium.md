# 🟡 Medium Dynamic Programming Problems

These problems introduce knapsack, subsequence, grid, and string DP patterns.

| # | Problem | Concept |
|---|---------|---------|
| 1 | Coin Change | Unbounded Knapsack |
| 2 | Coin Change II | Counting DP |
| 3 | Longest Increasing Subsequence | LIS |
| 4 | Longest Common Subsequence | String DP |
| 5 | Partition Equal Subset Sum | 0/1 Knapsack |
| 6 | Unique Paths | Grid DP |
| 7 | Decode Ways | String DP |
